# Contributing to Bact-Trace

Thank you for your interest in fighting Antimicrobial Resistance!

## How to Run Locally
1. Clone the repo: `git clone https://github.com/Deepak14Jain/bact-trace.git`
2. Set up the `.env` file (see README).
3. Run the Docker containers: `docker-compose up`

## Pull Request Process
1. Ensure your code passes the CI/CD build checks.
2. Update the README.md with details of changes to the interface.
3. You may merge the Pull Request in once you have the sign-off of two other developers.
