# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability
As this is a healthcare-focused application handling diagnostic data, we take security seriously.

If you discover a vulnerability, please DO NOT open a public issue.
Instead, email the core team at **jaindeepak1401+bact-trace@gmail.com**.
We will address the issue within 48 hours.
